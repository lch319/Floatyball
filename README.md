# Floatyball 🎈一款高度可自定义的多功能悬浮工具，支持动作快捷触发、文件多重拖放与动态时间显示。


> 一个悬浮球，万千种玩法。拖拽、吸附、透明、自定义动作、时间显示、GIF 动画……  
> 你的桌面，从此多一个得心应手的小精灵。

---


| 特性 | 说明 |
|------|------|
| 🖼️ **双形态切换** | 既可以是精致的 **PNG / GIF 悬浮球**，也可以是**纯文字时间条**，一键切换 |
| 🧲 **边缘智能吸附** | 拖到屏幕边缘自动贴边，鼠标离开后悄悄隐藏，靠近即现 |
| 🎚️ **动态透明度** | 鼠标悬停时高清显现，离开后温柔淡出，不遮挡工作内容 |
| 🔧 **万能自定义事件** | 左键、中键、滚轮、拖放文件/文本……每个操作都可绑定**6种动作类型** |
| 🎮 **修饰键组合** | 同一个点击，按住 `Ctrl` / `Alt` / `Shift` 可触发**完全不同的功能** |
| 🔄 **拖放 + 选中增强** | 直接拖文件/文字到球上执行操作；配合“获取选中内容”占位符，选中即用 |
| 📜 **强大占位符引擎** | 支持循环遍历、文件路径拆分、环境变量、鼠标位置……脚本能做的，它都能传 |
| 🧩 **可视化配置面板** | 所有设置都有暗色主题 GUI，修改实时生效，无需手写代码 |
| 💾 **配置备份与切换** | 一键保存当前配置、随时恢复历史备份，多套方案随心换 |

## 🚀 核心功能预览

### 1. 悬浮球主体
- 支持 **PNG / GIF**（自动播放动画）
- 自动识别图像透明区域，边缘吸附精准贴合
- **时间模式**：自定义字体、颜色、圆角背景、格式（支持换行）
- 滚轮缩放大小，最小 20px，最大 300px
- 精美关闭按钮，可显示/隐藏/调整样式

### 2. 事件动作系统
每个交互可独立配置 **4 种修饰状态**（无修饰、Ctrl、Alt、Shift），每种状态可选择：
- 🔹 运行程序 / 打开文件夹
- 🔹 发送按键
- 🔹 发送文本
- 🔹 调用 RunAny
- 🔹 内部命令（重载/退出）
- 🔹 执行自定义 AHK 代码

### 3. 拖放 & 占位符生态
```
将文件拖到球上 → 自动批量重命名、压缩、上传……
拖入文本 → 自动保存、翻译、搜索……
选中文件/文字 → 按一下快捷键，内容直接喂给球
```
内置 **20+ 占位符**，覆盖文件路径拆分、多行循环、动态变量、用户输入等。

### 4. 全局设置中心
通过 **暗色无边框 GUI** 一站式管理：
- 基础与外观（大小、热键、托盘图标）
- 行为与隐藏（吸附距离、全屏隐藏、固定位置）
- 透明度控制（基础值、渐变速度、延迟）
- 时间模式（格式、字体、颜色、背景）
- 关闭按钮（位置、大小、粗细、消失延迟）
- 配置备份（保存/加载方案、自动备份数量）

### 5. 便捷功能
- 全局热键显示/隐藏悬浮球
- 全屏应用（游戏、视频）时自动隐藏
- 编辑模式：方向键精确移动、`+/-` 调节大小
- 托盘图标支持左键恢复、右键菜单
- 开机自启、管理员权限运行

## 📦 快速上手
1. 下载最新版 `Floatyball.exe` 或克隆源码。
2. 将你的 `PNG/GIF` 图标放入 `Icons` 文件夹。
3. 运行程序，**右键悬浮球** 进入菜单，一切皆可配。
4. 享受高效、优雅的桌面操作体验！
